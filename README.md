# Bella Vista Restaurant Website

A fully responsive restaurant website built with HTML, CSS, and JavaScript.

## Features
- Responsive design (mobile, tablet, desktop)
- Mobile-friendly navigation with hamburger menu
- Interactive reservation form
- Menu showcase with hover effects
- Contact information and location
- Newsletter subscription

## Technologies Used
- HTML5
- CSS3 (Flexbox, Grid, Media Queries)
- Vanilla JavaScript
- Font Awesome Icons
- Google Fonts

## How to View
1. Clone this repository
2. Open `index.html` in your browser
3. Or view live at: https://github.com/Wilfred123816/wilfredmonyenye.github.io

## Project Structure
- `index.html` - Main HTML file
- `style.css` - All styling
- `script.js` - Interactive functionality
- `README.md` - Project documentation
